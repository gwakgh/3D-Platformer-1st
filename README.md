
# UE5 3D Platformer Game

🎮 연습 및 과제로 제작한 아주 간단한 언리얼 엔진 5 3D 플랫포머 게임입니다.  
블루프린트 기반으로 캐릭터 점프, 이동, 장애물, 컷신 등을 구현했습니다.

- **엔진 버전:** Unreal Engine 5.5.4
- **개발 방식:** 블루프린트

- **게임 장르:** 3D Platformer
